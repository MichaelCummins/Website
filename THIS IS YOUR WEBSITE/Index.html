The Cummins Compiler

Project link Source code

The Cummins Compiler is a recursive descent parser based on a LL(1) grammar. The Compiler goes through each step of the compilation process: lexing, parsing, semantic analysis, and code generation. The UI of the compiler also offers a view of the Abstract Syntax Tree and Concrete Syntax Tree located near the bottom of the page along with the machine code. Created design using HTML and CSS. Compiler developed using JavaScript.


Password Validator

Source code Project Link

This password validator is a java program that uses a Definite Finite Automata (DFA) to determine whether input from the user is a strong or weak password. The criteria used in constructing the DFA were: at least 1 uppercase letter, 1 lowercase letter, 1 digit, and 1 special character. This program is an easy to use application that enables users to choose the best password in order to enhance their personal security. Created using only Java.
